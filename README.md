# 🚗 GG Auto Vendita - Portale Web

Un'applicazione web leggera e reattiva progettata per la gestione e l'esposizione di un parco auto per la compravendita di veicoli.

---

## 📝 Descrizione del Progetto
GG Auto Vendita è una piattaforma vetrina sviluppata per offrire ai potenziali acquirenti un'interfaccia intuitiva per sfogliare annunci automobilistici. Il progetto punta sulla velocità di caricamento e su un design *mobile-first*, fondamentale per gli utenti che consultano i prezzi in mobilità.

Il sito è attualmente ospitato su: [https://www.ggautovendita.altervista.org/](https://www.ggautovendita.altervista.org/)

## ✨ Caratteristiche Principali
- **Catalogo Dinamico:** Visualizzazione delle auto con schede dettagliate (prezzo, chilometraggio, anno).
- **Design Responsive:** Ottimizzato per desktop, tablet e smartphone.
- **Ottimizzazione SEO:** Struttura pensata per il posizionamento locale sui motori di ricerca.

## 🛠 Tecnologie Utilizzate
Il progetto è stato realizzato utilizzando le seguenti tecnologie:
- **Frontend:** HTML5, CSS3
- **Scripting:** JavaScript (per filtri e gallerie immagini)
- **Hosting:** Altervista
